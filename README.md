# Zimi - Rede Social

#  Tecnologias Utilizadas
- Frontend: React, TypeScript, Redux
- Backend: Node.js, Express
- Banco de Dados: MongoDB
# Zimi é uma rede social onde os usuários podem se conectar e compartilhar suas experiências.
